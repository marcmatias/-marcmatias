Como atualizar meu fork com rebase no repositório origin

Posted 3. April 2021. 1 min read.

Atualizando fork com rebase origin repository

Guia com os comandos necessários para atualizar o fork com comando rebase.

  1. Set o upstream caso já não o tenha setado 
git remote add upstream https://github.com/original-repo/goes-here.git
  1. Pegue todas as modificações e branches do upstream 
git fetch upstream
  1. Com rebase reescrevemos a nossa master com o que foi baixado no passo anterior
git rebase upstream/master
  1. Agora é pushar as mudanças do clone local para o repositório fork no GitHub
git push origin master
  1. Na página do respositório fork no github será possivel ver a informação de que ele agora está em estado de even (in sync), ou seja, sincronizado com o repositório original.
